## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Export static site for GitHub Pages
npm run export
```

## Deployment to GitHub Pages

1. Update `next.config.js` with your repository name if needed
2. Build and export: `npm run export`
3. Push the `out` folder to the `gh-pages` branch, or use GitHub Actions
